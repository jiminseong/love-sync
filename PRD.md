# 📄 PRD: 뽀뽀뽀

**프로젝트 명:** 뽀뽀뽀
**한 줄 소개:** 화면에 뽀뽀하면 사랑 점수 알려주는 장난감 같은 웹 실험
**해커톤 목표:** 2시간 30분 안에 Landing → Scan → Report 풀 플로우 구현

---

## 1. 컨셉
화면에 가볍게 뽀뽀하면, 카메라랑 마이크가 입술 거리·안정성·소리를 읽어서 **사랑 점수(♥%)** 와 짧은 코멘트를 즉시 발행한다. 무겁고 학술적인 톤은 다 빼고, 친구가 장난치듯 말 거는 가벼운 톤으로 — "와, 진심도 92점이네 ♥" 같은 느낌.

심오한 메시지나 사회 풍자는 의도적으로 빼고, **재밌고 빠른 SNS용 장난감**에 집중한다.

## 2. 타겟
- 친구들과 같이 깔깔거리며 한 번씩 해보고 캡처해서 SNS에 올릴 2030.
- 머무르는 시간 1~2분, 결과 캡처 → 공유까지 30초 안에 끝나야 한다.

## 2.1. 프라이버시 원칙 (절대 양보 X)
- **미디어 데이터(카메라/마이크/캡처 이미지)는 외부 송신 0.**
- MediaPipe·Web Audio 모두 클라이언트 추론.
- Gemini로는 점수·메트릭 텍스트만 송신하고, 응답으로 처방전·finale 텍스트만 수신한다.
- 사용자가 "저장"을 눌렀을 때 생성되는 결과 카드 이미지는 로컬 다운로드만 수행한다.
- 화면의 *"안심해, 다 네 폰 안에서만 일어나."* 슬로건은 미디어 데이터에 한정해 유효하다는 점을 report/finale 고지에서 분명히 한다.

---

## 3. 핵심 기능

### 3.1. 뽀뽀 측정 (Vision + Audio)
- **입술 추적:** MediaPipe Face Mesh로 입술 랜드마크 추출 → 화면 대비 거리 + 흔들림 안정성 계산.
- **소리 분석:** Web Audio `AnalyserNode`로 RMS/피크 dB → "쪽" 어택 감지.
- **점수:** 거리 안정성과 소리를 가중합해서 0~100 사이 사랑 점수.

### 3.2. 결과 카드
- 큰 점수 한 줄 (예: **87 ♥**)
- 짧은 코멘트 1~2줄 (가벼운 농담 톤). 예: *"오, 진심 좀 있는데?"*
- 미니 메트릭 3개 정도 (거리/안정성/소리) — 표가 아닌 친근한 라벨로
- "다시 하기" / "저장하기" 버튼

학술 캡션, 일련번호, REPORT NO. 같은 진지한 장식은 **모두 제거**.

### 3.3. AI 결과 보강(선택, 폴백 가능)
- Report 진입 직후 Gemini 2.5 Flash가 점수·성향·거리 안정성·피크 dB·기기명만 받아 처방전과 finale 문장을 보강한다.
- 응답 지연, 네트워크 실패, 스키마 불일치가 있으면 기존 정적 처방전과 finale 문장을 그대로 유지한다.
- 어떤 경우에도 카메라 프레임, 오디오 버퍼, 캡처 이미지는 Gemini나 Vercel Function으로 보내지 않는다.

---

## 4. 유저 플로우
1. **Landing:** "내 기기, 지금 사랑이 고파 ♥" + "지금 뽀뽀하러 가기 →" 버튼.
2. **Scan:** 클릭 즉시 카메라+마이크 권한을 한 번에 요청. 라이브 미리보기 + 입술 가이드를 보면서 사용자가 화면에 가까이 다가가 "쪽" 소리를 낸다. 시스템은 매 프레임 **(a) 입술 근접도(proximity)** 와 **(b) "쪽" 어택(dB threshold + 짧은 attack)** 을 동시에 감시하다가 둘 다 충족되는 순간 자동으로 분석을 마감한다. 사용자가 누를 별도 "분석 시작" 버튼은 없다.
3. **Report:** 점수 카드 + 코멘트 + 저장/다시 하기.

별도의 "Action" 진행바 화면은 두지 않는다 — Scan 안에서 짧은 fade transition만으로 곧장 Report로 넘어가게 해서 흐름을 끊지 않는다.

---

## 5. 기술 스택
- **Frontend:** HTML/CSS/Vanilla JS, 빌드 도구 없음.
- **Vision:** MediaPipe Face Mesh CDN.
- **Audio:** Web Audio API.
- **AI:** Gemini 2.5 Flash.
- **Backend:** Vercel Functions(Node.js 22), `GOOGLE_API_KEY` 서버 환경변수.
- **Report:** JS 템플릿 배열 조합 + Gemini 텍스트 보강(실패 시 정적 폴백).
- **캡처:** html2canvas로 이미지 저장.
- **배포:** Vercel 정적, HTTPS 필수.

## 6. 톤 & 보이스
- 친구가 장난치듯, 짧고 가볍게.
- 존댓말보다 반말 또는 부드러운 말끝(어/네/♥).
- "분석 중", "진단", "리포트", "실험", "센서 교정" 같은 딱딱한 단어 **사용 금지**.
- 대신 "측정 중", "체크", "결과", "준비" 같은 일상어.
- 이모지/하트는 화면당 1~2개까지 가벼운 강조에 한정.
